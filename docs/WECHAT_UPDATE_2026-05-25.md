## 1. 数据扩容 — 真实商品，**不再 AI 生成**

> 经李雨晟反馈，前两轮的数据都是 AI 生成，缺少答辩可信度。Round 6 全部停止合成，改为**真实商品搜索**。

- 两个并行 research agent 跑了 ~15 分钟，从 **京东、Amazon US、Amazon JP** 抓回了 **45 个真实商品**，每个都带：
  - 真实商品详情页 URL（`external_url`）— 点开就能跳到 Tmall/JD/亚马逊
  - 真实价格 + 币种（CNY / USD）
  - 真实平台 CDN 上的商品图
  - 来源国家（CN/US/JP/DE/FR）+ 来源平台 + 运输备注（国内现货 / 海外直邮）
- 商品目录从 **100 → 145**，类目从 **4 → 8**（新增母婴健康 / 家居家具 / 图书音像 / 户外运动）。
- 用户问"推荐一本书"现在能命中 `7_图书音像`（三体、活着、平凡的世界…）。
- 老的 100 个 AI 生成商品**保留**，UI 上贴一个**「演示」徽章**，让评委一眼看出哪些是真、哪些是 demo。

详细方法 + 局限性：[`docs/research/2026-05-24-real-products.md`](research/2026-05-24-real-products.md)。

## 2. 多币种 / 多产地 — UI 直接区分

- 商品卡片左上角：国旗 emoji（🇨🇳/🇺🇸/🇯🇵/🇩🇪/🇫🇷）
- 价格行：货币符号 + 中文括注，`$95 (美元)` 不会被误读成 95 块
- 品牌行加来源平台前缀：`Amazon US · Sony` / `JD · 雅诗兰黛`
- 购物车 / 结算：按币种分组汇总，**不做汇率折算**（演示阶段诚实)
  - 例如：`合计：¥1,200 + $35（跨境订单不做实时汇率合计）`

## 3. UI 体验补刀

- ⌛ **会话等待时来一句段子**（评委说慢的痛点）：
  - 「🦁 狮子小哥正在认真比价中…」
  - 「📦 翻箱倒柜帮你找好物…」
  - 「🦴 别催，狮子在啃骨头思考」
  - 12 句循环，每 1.5 秒切换一次。第一个 delta 到达时立刻被替换成正文。
- 🛒 **聊天里的商品卡片支持一键加购**（右上 `+` 小药丸）：
  - 不影响点击卡片正文进入详情；
  - 加购有触觉反馈 + 「已加入」浮层 → 用户立刻感知。
- 🗑 **购物车支持显式删除**（垃圾桶按钮 + 编辑模式）：
  - 原来只能左滑，新人发现不了。
- 🌐 **「去原页」按钮**：详情页直接跳 Tmall / JD / Amazon。演示商品则灰显，文案："演示商品 · 无原页链接"（不糊弄）。
