# meetings/

Meeting notes live here. One file per meeting.

**File name**: `YYYY-MM-DD-<topic-slug>.md`. Examples:
- `2026-05-20-kickoff.md`
- `2026-05-29-rag-eval-review.md`
- `2026-06-08-dress-rehearsal.md`

Use [docs/meeting_template.md](../docs/meeting_template.md) as the starting point.

Action items in meeting notes should be checkboxes with an owner and a due date. They migrate to [ROADMAP.md](../docs/ROADMAP.md) if they span more than a few days.
